Using Maven

Download and install Maven.

mvn archetype:generate \\

-DinteractiveMode=false \\

-DarchetypeGroupId=org.apache.maven.archetypes \\

-DarchetypeArtifactId=maven-archetype-quickstart \\

-DarchetypeVersion=1.1 \\

-DgroupId=fr.lirmm.graphik \\

-DartifactId=getting-started \\
          
-Dversion=1.0-SNAPSHOT

Add the code snippet below into the pom.xml file between the <dependencies> and </dependencies> tags.

<dependency>
    <groupId>fr.lirmm.graphik</groupId>
    <artifactId>graal</artifactId>
    <version>__GRAAL_VERSION__</version>
</dependency>

Edit the main method in the file "src/main/java/fr/lirmm/graphik/App.java".
