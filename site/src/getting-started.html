Getting Started

#include "nav.html"

Requirements

  • Graal requires Java 7 or newer.
  • Download Graal
  • Download Example0.java
  • Download animals.dlp

Build

Open a terminal and go to the folder where downloaded files are. Then, execute the command below:

javac -cp .:./graal-__GRAAL_VERSION__.jar Example.java

Alternatively, you can download the pre-compiled version.

Run the example

Execute this command:

java -cp .:./graal-*.jar Example

What does this example do?

  1. Create a relational database store with HSQLDB (An InMemory Java database system),
  2. Parse Animals.dlp (A __DLGP__ file with rules and facts),
  3. Print the set of rules in __DLGP__,
  4. Print the set of facts in __DLGP__,
  5. Parse a query from the code and print it,
  6. Query the store without reasoning,
  7. Backward chaining,
  8. Forward chaining.
