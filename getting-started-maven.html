This example shows how to build and run a simple Graal project using the Maven build tool. If you prefer, you can build and run this example without any build tool.

Requirements

  • Graal requires Java 7 or newer.
  • Download and install Maven.
  • Download the project graal-getting-started.zip and extract it where you want.

Build

Open a terminal and go to the graal-getting-started folder. Then, execute the command below:

mvn package

Run the example

Execute the project:

java -jar target/getting-started-1.0-SNAPSHOT.jar src/main/resources/animals.dlp

What does this example do?

  1. Create a relational database store with HSQLDB (an InMemory Java database system),
  2. Parse Animals.dlp (a Dlgp file with rules and facts),
  3. Print the set of rules in Dlgp format,
  4. Print the set of facts in Dlgp format,
  5. Parse a query from the code and print it,
  6. Query the store without rule reasoning,
  7. Backward chaining (based on query rewriting),
  8. Forward chaining (based on fact saturation).
