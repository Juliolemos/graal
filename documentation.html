Overview

Graal is dedicated to ontology-mediated query answering. We consider a conjunctive query over a knowledge base composed of an ontology and a set of facts. The ontology is described in the framework of existential rules, aka Datalog+/-.

Graal can be broken down into seven parts:

  • The Core module implements main classes such as Atom, AtomSet, Rule, ConjunctiveQueryâ€¦
  • The IO module provides readers from or writers to textual formats.
  • The Store module allows access to storage systems.
  • The Homomorphism module implements algorithms to query data.
  • The Forward Chaining module implements forward chaining (aka chase) algorithms.
  • The Backward Chaining module implements query rewriting algorithms.
  • The Rule Set Analyser module provides tools to analyse decidability properties of a rule set.

Several tools are built over Graal:

Pure
A query rewriter.
OWL2DLGP
A translator from OWL2 to Dlgp.
DLGP2RuleML
A translator from Dlgp to the Datalog+ fragment of Deliberation RuleML 1.01.
Kiabora
An analyser of a set of existential rules.
Utility tools
A set of utilities to manage rules.
