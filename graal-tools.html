The graal-tools program is a set of utilities to manage rules. It currently has three features:

atomic-head
transform a rule set into a set of atomic-head rules.
singlepiece-head
transform a rule set into a set of singlepiece-head rules.
labeler
add a label to each rule.

Getting started

  • Download graal-tools
  • Download example1.dlp

Compute atomic-head

java -jar graal-tools-*.jar -a < ./example1.dlp

Compute singlepiece-head

java -jar graal-tools-*.jar -p < ./example1.dlp

Labels rules

java -jar graal-tools-*.jar -l < ./example1.dlp
