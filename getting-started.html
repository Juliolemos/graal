This page shows how to build and run a simple Graal project without any build tool. If you prefer, you can build and run this example using Maven.

Requirements

  • Graal requires Java 7 or newer.
  • Download Graal (this package contains the core, homomorphism, backward_chaining, forward_chaining, io-dlgp, store-rdbms modules and their dependencies)
  • Download Example0.java
  • Download animals.dlp

Build

Open a terminal and go to the folder where downloaded files are. Then, execute the command below:

javac -cp .:./graal-1.1.0-for-example0.jar Example0.java

Alternatively, you can download the pre-compiled version.

Run the example

Execute this command:

java -cp .:./graal-1.1.0-for-example0.jar Example0 animals.dlp

What does this example do?

  1. Create a relational database store with HSQLDB (an InMemory Java database system),
  2. Parse Animals.dlp (a Dlgp file with rules and facts),
  3. Print the set of rules in Dlgp format,
  4. Print the set of facts in Dlgp format,
  5. Parse a query from the code and print it,
  6. Query the store without rule reasoning,
  7. Backward chaining (based on query rewriting),
  8. Forward chaining (based on fact saturation).
