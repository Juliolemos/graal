OWL2 to Dlgp

Home Documentation Publications Downloads Sources
The owl2dlgp program translates to dlgp any OWL2 ontologies that can be expressed into the existential rule framework. In particular, it translates the 3 tractable profiles of OWL2 (EL, QL, RL). This tool uses the OWL API to parse OWL2 ontologies that can be expressed in OWL/XML, OWL Functional Syntax or Turtle.

Getting started

Download owl2dlgp.

  • Translate from OWL file to dlgp file: 
    java -jar owl2dlgp-*.jar -f ./example.owl -o ./example.dlp
  • Translate from the standard input to the standard output: 
    java -jar owl2dlgp-*.jar < ./example.owl > ./example.dlp
